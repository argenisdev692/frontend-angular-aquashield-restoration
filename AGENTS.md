# This is NOT the Angular you know

This version has breaking changes — APIs, conventions, and file structure may all differ from your training data. Angular 22 (June 2026) requires TypeScript 6, uses Standalone Components by default, Signals for state management, and native control flow. OnPush is now the default change detection strategy. `resource()`, `rxResource()`, `httpResource()`, and Signal Forms are all stable. Read the official Angular.dev documentation before writing any code. Heed deprecation notices.
